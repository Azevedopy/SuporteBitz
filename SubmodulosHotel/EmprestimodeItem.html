Como Excluir Empréstimo de Item

Guia completo para excluir/devolver itens emprestados vinculados a uma UH no sistema Bitz

Para excluir um empréstimo vinculado a uma UH (Unidade Habitacional), siga o passo a passo detalhado abaixo:

1 Acesso ao Painel
2 Localizar UH
3 Informações do Empréstimo
4 Excluir/Devolver

Passo 1: Acesso ao painel de UH

Vá até o menu lateral esquerdo do sistema Bitz. Clique em "Painéis" ou acesse diretamente pelo link:

PAINEL DE UH'S

Esta é a tela principal onde você visualiza e gerencia todas as unidades habitacionais do hotel.

Passo 2: Localizando a UH

Na tela de painéis, localize a UH (quarto) para a qual deseja excluir o empréstimo do item.

Acesso ao quarto

Ao acessar o quarto, você será direcionado automaticamente para a aba "Informações Gerais". Ao lado, procure o ícone de uma cesta com o nome "Objetos Emprestados" e clique sobre ele.

Dica

Role a tela para baixo até visualizar os produtos emprestados a essa UH.

Passo 3: Informações exibidas dos empréstimos

Cada item emprestado será exibido com as seguintes informações:

Detalhes do Empréstimo
  • Código do Empréstimo
  • Descrição do Produto
  • Observações do Empréstimo (logo abaixo da descrição)
  • Quantidade de Itens
  • Hóspede Responsável pelo Empréstimo
  • Usuário do Sistema que fez o Lançamento
  • Data de Devolução (caso o item já tenha sido devolvido)
Ações disponíveis

As ações são apresentadas por quatro botões:

Devolver

Cobrar Item

Marcar Item Perdido

Imprimir Comprovante

Passo 4: Como realizar a exclusão do empréstimo

Para realizar a "exclusão" do empréstimo, utilize a opção "Devolver".

  • Clique em "Devolver" no item que deseja excluir
  • Confirmação - Será exibida uma caixa de solicitação com a seguinte pergunta:
    "Deseja devolver o item: [NOME DO ITEM]?"
  • Marque como Sim para confirmar a devolução
  • Item devolvido - O item será automaticamente devolvido, removendo-o da lista de itens emprestados da UH
    • Empréstimo excluído com sucesso! O item foi devolvido e removido da lista de empréstimos ativos.

    Importante

    No sistema Bitz, a "exclusão" de um empréstimo é feita através da funcionalidade "Devolver". Isso mantém o histórico no sistema mas remove o item da lista de empréstimos ativos.
