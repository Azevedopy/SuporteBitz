Configurações de Estoque

Guia completo para parametrizar o controle de estoque no sistema BITZ All In One

Este documento tem como finalidade orientar os usuários do sistema BITZ All In One sobre o acesso e configurações da tela Configurações de Estoque. Essa tela possui o objetivo de parametrizar o estoque e suas permissões.

Passo 1: Como Acessar as Configurações

Para acessar as configurações de estoque, siga um dos caminhos abaixo:

  • Menu lateral esquerdo: Cadastros > Operacional > Configurações de Estoque
  • Link direto: https://hotel.bitzsoftwares.com.br/sistema/configuracoes/estoque/

    • Acesso às configurações: Após acessar a tela, clique no botão "Editar" no lado direito para acessar as configurações disponíveis.

    Passo 2: Controle de Estoque

    Estes parâmetros controlam os níveis e limites do estoque no sistema:

    Campo Descrição
    Utiliza Estoque Mínimo

    Define se o sistema deve controlar o nível mínimo de estoque para cada produto.

    Sim
    Não

    Sim: o sistema emitirá alertas ou bloqueios quando o estoque atingir o limite mínimo cadastrado.

    Não: o sistema não controla níveis mínimos e permite consumo livre, mesmo que o item esteja quase acabando.
    Utiliza Estoque Negativo

    Determina se o sistema permite movimentar ou lançar produtos mesmo sem saldo disponível.

    Sim
    Não

    Sim: o estoque pode ficar negativo (ex: -2 unidades), útil para ajustes temporários.

    Não: o sistema bloqueia lançamentos se não houver quantidade suficiente.
    60%

    Importante

    O controle de estoque mínimo é fundamental para:

    • Alertas preventivos para reabastecimento
    • Evitar ruptura de estoque de itens essenciais
    • Planejamento de compras baseado em consumo real
    • Notificações automáticas para o setor de compras

    Passo 3: Baixa Automática na Comanda

    Campo Descrição
    Baixar Estoque pela Comanda

    Indica se o sistema deve dar baixa automática no estoque quando um produto for lançado na comanda do hóspede.

    Sim
    Não

    Sim: a baixa ocorre no momento do lançamento na comanda.

    Não: a baixa é feita manualmente ou apenas quando a conta é encerrada.

    Baixa Automática

    Estoque atualizado em tempo real ao lançar produtos na comanda.

    Sincronização

    Vendas e estoque sempre sincronizados sem necessidade de ajustes manuais.

    Controle Preciso

    Evita divergências entre o estoque físico e o sistema.

    Relatórios Confiáveis

    Dados de consumo sempre atualizados para análise.

    Recomendação

    Para a maioria dos hotéis e restaurantes, recomenda-se configurar como "Sim" para garantir que o estoque seja atualizado automaticamente sempre que um produto for consumido. Isso mantém o controle de estoque sempre preciso e atualizado.

    Passo 4: Configurações Locais e Contábeis

    Campo Descrição
    Local de Estoque Padrão Define qual depósito, almoxarifado ou setor será utilizado por padrão para movimentações e baixas.
    Obrigatório Informar Centro de Custo

    Determina se é obrigatório vincular cada movimentação de estoque a um centro de custo.

    Sim
    Não

    Sim: o sistema exige o preenchimento (ex: Hospedagem, Alimentos e Bebidas, Lavanderia).

    Não: o campo é opcional.
    Obrigatório Informar Analítica de Receita Se ativo, obriga o usuário a informar a conta contábil de receita detalhada associada ao item vendido.
    Obrigatório Informar Analítica de Despesa Semelhante ao campo anterior, mas aplicado às despesas.
    Exibe Saldo Atual dos Produtos Define se o sistema deve mostrar o saldo em estoque durante o lançamento ou consulta.
    Listar Somente Produtos Vinculados ao Local de Estoque Define se o sistema deve exibir apenas os produtos cadastrados para o depósito/local selecionado.

    Configurações Contábeis

    As configurações de centros de custo e contas analíticas são essenciais para:

    • Precisão contábil nas movimentações de estoque
    • Análise de custos por departamento/área
    • Rateio adequado de despesas e receitas
    • Auditoria facilitada com rastreamento completo

    Controle de Acesso

    Define níveis de permissão para diferentes usuários.

    Rastreabilidade

    Registra todas as alterações com justificativas quando necessário.

    Segurança

    Protege contra alterações não autorizadas no estoque.

    Auditoria

    Facilita auditorias internas e externas do estoque.

    Configurações de estoque ajustadas com sucesso! O sistema agora controlará o estoque conforme os parâmetros definidos, garantindo maior precisão e segurança nas movimentações.

    Recomendações de Segurança

    Para maior segurança no controle de estoque, recomenda-se:

    • Exigir senha para exclusões de produtos lançados
    • Exigir justificativa para edições de diárias
    • Limitar permissões apenas a usuários autorizados
    • Manter histórico completo de todas as alterações
    • Configurar alertas para movimentações críticas
