Alterar Senha para Estorno/Exclusão

Guia completo para configurar senhas de segurança para operações financeiras

Configure senhas específicas para autorizar operações sensíveis como estorno de lançamentos e exclusão de pagamentos, garantindo maior segurança no controle financeiro.

Passo 1: Acessar Parâmetros Financeiros

Para alterar as senhas de estorno e exclusão, siga o caminho no sistema:

Caminho de Acesso

Menu lateral → "Cadastros" → "Cadastros Auxiliares (Financeiro)" → "Parâmetros Financeiros"

Acessar Parâmetros Financeiros

Parâmetros Financeiros

Neste menu, procure por "Ações" e clique no ícone de "Lápis em Azul" para editar as configurações existentes nos parâmetros.

Editar Parâmetros

Configurar Senhas

Segurança

Permissões Necessárias

Para acessar e modificar os parâmetros financeiros, você precisa ter permissões de administrador ou gerente financeiro no sistema.

Passo 2: Configurar Senha de Estorno

Senha para Estorno de Lançamento

Localizar Configuração

Procure pela opção: "Validar Senha no Estorno de Lançamento?"

Validar Senha no Estorno de Lançamento?
Sim
Não

Se estiver marcado "Sim", uma barra Azul com o ícone de um "Lápis" aparecerá escrito "Editar Senha - Edição Lançamento".

Alterar Senha

Clique no botão azul para abrir o formulário de alteração de senha e:

Nova Senha:
Repetir Senha:
  • Altere para a senha desejada seguindo as políticas de segurança
  • Repita essa senha no campo de confirmação
  • Use senhas fortes com letras, números e caracteres especiais
  • Mantenha a senha segura e compartilhe apenas com pessoas autorizadas

Segurança Financeira

Protege contra estornos não autorizados de lançamentos financeiros.

Controle de Alterações

Registra quem realizou estornos e quando foram feitos.

Auditoria

Facilita auditorias financeiras com controle de acessos.

Prevenção de Erros

Evita estornos acidentais por colaboradores não autorizados.

Passo 3: Configurar Senha de Exclusão

Senha para Exclusão de Lançamentos (Pagamentos)

Localizar Configuração

Procure pela opção: "Validar Senha na Edição de Lançamento (Pagamentos)?"

Validar Senha na Edição de Lançamento (Pagamentos)?
Sim
Não

Se estiver marcado "Sim", uma barra Azul com o ícone de um "Lápis" aparecerá escrito "Editar Senha - Edição Lançamento".

Alterar Senha

Clique no botão azul para abrir o formulário de alteração de senha e:

Nova Senha:
Repetir Senha:
  • Altere para a senha desejada (pode ser a mesma do estorno ou diferente)
  • Repita essa senha no campo de confirmação
  • Diferencie as senhas para maior segurança (recomendado)
  • Controle o acesso a pessoas específicas da equipe financeira

Boas Práticas de Segurança

  • Senhas diferentes para estorno e exclusão aumentam a segurança
  • Altere as senhas periodicamente (a cada 90 dias recomendado)
  • Mantenha registro das alterações de senha realizadas
  • Compartilhe apenas com pessoas que realmente precisam ter acesso
  • Documente as políticas de uso das senhas na empresa

Passo 4: Salvar Configurações

Resumo do Processo

  • Acesse: Menu lateral → Cadastros → Cadastros Auxiliares (Financeiro) → Parâmetros Financeiros
  • Localize: Ações e clique no Lápis Azul para editar
  • Para estorno: Configure "Validar Senha no Estorno de Lançamento?" e defina a senha
  • Para exclusão: Configure "Validar Senha na Edição de Lançamento (Pagamentos)?" e defina a senha
  • Clique em "Salvar" no botão em azul para aplicar as alterações
    • Finalizar Configuração

    Após configurar ambas as senhas, vá até o final da página e localize o botão de salvar:

    Clique em "Salvar" para que todas as alterações sejam aplicadas no sistema.

    Configuração Aplicada

    As senhas estarão ativas imediatamente após o salvamento.

    Histórico

    O sistema registra quando e quem alterou as configurações.

    Atualização Imediata

    As alterações são aplicadas em tempo real em todo o sistema.

    Notificações

    Administradores podem ser notificados sobre alterações nas senhas.

    Senhas configuradas com sucesso! Agora as operações de estorno e exclusão de lançamentos exigirão autorização por senha, aumentando a segurança financeira do estabelecimento.

    Teste as Configurações

    Após salvar as configurações, recomenda-se testar o funcionamento:

    • Tente estornar um lançamento de teste para verificar se solicita senha
    • Tente excluir um pagamento de teste para confirmar a solicitação de senha
    • Verifique com diferentes usuários se as permissões estão corretas
    • Documente as senhas em local seguro seguindo políticas de segurança
