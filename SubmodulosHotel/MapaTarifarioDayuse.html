Mapa Tarifário Day Use

Guia completo para administrar e ajustar valores das tarifas de Day Use

O Mapa Tarifário Day Use é o local onde o hotel administra e ajusta os valores das tarifas de Day Use ao longo do calendário. Essa funcionalidade permite definir preços específicos por dia, por faixa etária, por serviço e por período.

Para acessar o módulo de Mapa Tarifário do Day Use, siga o caminho no sistema:

Passo 3: Atualizando Preços no Mapa Tarifário

Para atualizar os preços no mapa tarifário, clique no botão "Atualizar Preços" no cabeçalho. Isso abrirá uma janela com os seguintes campos:

A

Data Inicial e Data Final

Define o período a ser atualizado. Você pode aplicar valores para um único dia ou para um intervalo de datas.

B

Seleção da Tarifa

Lista todas as tarifas cadastradas para edição. Selecione a tarifa que deseja atualizar.

C

Seleção dos Dias da Semana

Permite aplicar valores apenas em dias específicos (ex.: somente finais de semana).

Passo 4: Exemplos Práticos

Aqui estão exemplos práticos de como utilizar o Mapa Tarifário:

Exemplo 1: Café + Almoço

Configuração:

  • Período: 17/11/2025 a 17/12/2025
  • Valor: R$ 250,00
  • Dias: Todos os dias
  • Tarifa: Café + Almoço

Resultado: Todos os dias entre 17/11 e 17/12 terão a tarifa "Café + Almoço" configurada em R$ 250,00.

Exemplo 2: Tarifa Adulto

Configuração:

  • Período: 01/12/2025 a 31/12/2025
  • Valor: R$ 120,00
  • Dias: Somente sábados e domingos
  • Tarifa: Adulto

Resultado: Apenas os finais de semana de dezembro terão a tarifa "Adulto" configurada em R$ 120,00.

Dica Prática

Use o Mapa Tarifário para criar promoções sazonais, aumentos de preço em feriados, ou descontos em períodos de baixa ocupação. Isso maximiza a receita e otimiza a ocupação.

Passo 5: Relação entre Tarifas e Mapa Tarifário

Entenda a diferença fundamental entre as duas funcionalidades:

Tarifas Day Use

Definem o tipo de tarifa

Criam as categorias de preços (Adulto, Criança, Café+Almoço, etc.)

Mapa Tarifário

Define quanto custa e quando custa

Aplica valores específicos para cada tarifa em datas específicas

Fluxo de Configuração
  • Primeiro, crie as tarifas em "Tarifas Day Use"
  • Depois, defina os valores no "Mapa Tarifário Day Use"
  • As reservas de Day Use usarão os valores definidos no mapa

    Passo 7: Resumo do Processo

    Resumo do Processo Completo

  • Acesse o Mapa Tarifário em Menu → Day Use → Mapa Tarifário Day Use
  • Escolha a data de referência e clique em "Carregar"
  • Visualize os valores atuais na grade tarifária
  • Clique no botão "Atualizar Preços"
  • Defina o período, selecione a tarifa, escolha os dias e insira o valor
  • Salve as alterações para aplicar os novos preços

    • Pronto para Uso

    Após salvar, as novas tarifas estarão disponíveis imediatamente para reservas.

    Atualização em Tempo Real

    Todas as mudanças são refletidas instantaneamente no sistema.
