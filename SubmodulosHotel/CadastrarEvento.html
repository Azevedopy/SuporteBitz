Visão Geral do Módulo de Eventos

O novo módulo de eventos do sistema Bitz Hotel permite a criação e gestão completa de espaços e eventos como reuniões, shows, conferências, coquetéis e palestras corporativas, conforme os espaços disponíveis no hotel.

Funcionalidades Principais
  • Cadastro completo de espaços com dimensões, valor estimado e status
  • Tipos de eventos personalizáveis com cores de status visuais
  • Tabelas de preços específicas com políticas de idade personalizadas
  • Criação de eventos completos com seleção de espaços e recursos
  • Mapa visual de eventos para acompanhamento em tempo real
  • Controle de status de negociação (Frio, Morno, Quente, Confirmado, Perdido)

Acesso ao Módulo de Eventos:

No menu lateral esquerdo, é possível encontrar a opção 'Eventos', onde o usuário pode estruturar todo o evento através dos seguintes cadastros:

  • Tipo de Espaço - Categorização dos espaços
  • Espaço - Características físicas dos espaços
  • Tipo de Evento - Classificação dos eventos
  • Tabela de Preços - Valores e políticas

Importante: Em todos os menus de cadastro e evento é possível criar e editar opções já criadas. O sistema oferece flexibilidade total para ajustes durante todo o processo.

Cadastros Básicos para Configuração

Antes de criar eventos, é necessário configurar os cadastros básicos que servirão como base para toda a gestão.

2.1 Tipo de Espaço

Neste menu, o usuário cadastra uma descrição para o tipo de espaço e define o status como ativo ou inativo.

Ativo Inativo

Exemplos: Salão de Festas, Auditório, Sala de Reunião, Área Externa

2.2 Espaço

Permite cadastrar as características dos espaços que o hotel possui, tais como dimensões (largura e comprimento), valor estimado, status de ocupação e descrição.

Neste campo também é selecionado o tipo de espaço cadastrado anteriormente.

2.3 Tipo de Evento

Permite cadastrar um tipo de evento que será utilizado posteriormente na criação de eventos. É possível definir o nome, cores de status (para classificação visual), status ativo/inativo e vincular uma tabela de preços.

Frio Morno Quente
2.4 Tabela de Preço

Neste menu, cadastra-se uma tabela de preços específica para o evento. A descrição corresponde ao nome da tabela, e os valores podem ser configurados por pessoa.

Também é possível criar políticas de idade específicas para o evento (ex: crianças até 5 anos gratuitas).

Sequência Recomendada de Cadastro
  1. 1. Tipo de Espaço: Crie as categorias de espaços disponíveis
  2. 2. Espaço: Cadastre cada espaço físico vinculando ao tipo correspondente
  3. 3. Tabela de Preços: Configure as tabelas de valores para diferentes tipos de eventos
  4. 4. Tipo de Evento: Crie os tipos de eventos e vincule às tabelas de preços

Seleção de Espaços

Ao lado de informações gerais tem a opção "Espaços" aonde é possível selecionar o espaço que deseja incluir nesse evento:

  1. Clique em "Adicionar espaço +"
  2. Abrirá uma janela com opções de tipo do espaço, formato do espaço e data/hora
  3. Clique para buscar disponibilidade
  4. Aparecerá os espaços disponíveis, basta selecionar
  5. Clique em salvar para incluir o espaço no evento
UH/Pessoas e Recursos
  • No menu 'UH/Pessoas': É possível cadastrar classificações e unidades habitacionais, além de definir o número de pessoas por categoria.
  • No menu 'Recursos': Funciona como o menu de extras da reserva, permitindo adicionar produtos e serviços adicionais como buffet, equipamentos e outros recursos necessários para o evento.

Observações importantes quanto ao mapa de evento:

  • Não é possível fazer alterações arrastando, assim como no mapa de ocupação. As edições devem ser feitas pelos menus de edição do evento.
  • É possível filtrar por status de negociação: FRIO, MORNO, QUENTE, CONFIRMADO, PERDIDO (PRETO), ABERTO (em andamento), ENCERRADO
  • É possível filtrar por data: O mapa mostra por dia, semana e mês
  • Visualização clara: Cada cor representa um status diferente, facilitando a identificação rápida
Este módulo traz flexibilidade e controle completo sobre a gestão de eventos, permitindo ao hotel otimizar o uso de seus espaços e serviços.

Jornada Completa do Usuário

O Módulo de Eventos do sistema Bitz Hotel permite que o usuário realize todo o processo de criação, gestão e acompanhamento de eventos dentro do hotel, desde o cadastro dos espaços até o encerramento do evento.

Jornada do Usuário no Módulo de Eventos
1
Acesso ao Módulo

O usuário acessa o menu lateral esquerdo do sistema e seleciona a opção 'Eventos'. Nesta área, é possível visualizar os submenus de Cadastro (Tipo de Espaço, Espaço, Tipo de Evento, Tabela de Preços), Evento (busca, edição e criação) e Mapa de Evento.

2
Etapa de Configuração Inicial (Cadastros Base)

Antes de criar um evento, o usuário deve realizar os cadastros base: Tipo de Espaço (descrição e status), Espaço (características físicas), Tipo de Evento (classificação e cores) e Tabela de Preços (valores e políticas).

3
Etapa de Criação do Evento

Após configurar os cadastros base, o usuário cria o evento em si, preenchendo descrição, solicitante, datas, tipo de evento e status de negociação (Frio, Morno, Quente, Confirmado ou Perdido).

4
Configuração Detalhada do Evento

O usuário seleciona espaços (com busca de disponibilidade), configura unidades habitacionais e pessoas (capacidades), e adiciona recursos e serviços adicionais (buffet, equipamentos, decoração).

5
Mapa de Evento - Visualização e Acompanhamento

O mapa de evento exibe todos os eventos registrados, permitindo filtrar por status de negociação e por data (dia, semana ou mês). Não é possível mover eventos arrastando, sendo necessário editar pelos menus.

6
Encerramento e Gestão Contínua

O evento pode ser alterado, cancelado ou encerrado conforme o andamento. O histórico e os status permitem analisar a performance e ocupação dos espaços, garantindo um controle completo do ciclo de vida do evento.

Benefícios da Jornada Estruturada
  • Processo organizado: Sequência lógica desde o cadastro básico até a gestão completa
  • Controle total: Todas as etapas do evento são gerenciáveis dentro do sistema
  • Visibilidade completa: Mapa de eventos oferece visão geral em tempo real
  • Flexibilidade: Status de negociação permite acompanhar a evolução dos eventos
  • Otimização de recursos: Melhor aproveitamento dos espaços e serviços do hotel

Conclusão e Melhores Práticas

O Módulo de Eventos do Bitz Hotel representa uma solução completa e integrada para a gestão de eventos corporativos e sociais, oferecendo controle total sobre espaços, recursos e acompanhamento.

Melhores Práticas Recomendadas
  • Complete todos os cadastros básicos antes de começar a criar eventos para garantir consistência nos dados
  • Use as cores de status estrategicamente para facilitar a identificação visual dos eventos
  • Sempre verifique disponibilidade antes de confirmar espaços para evitar conflitos
  • Atualize os status de negociação conforme o evento evolui para manter informações precisas
  • Consulte o mapa de eventos regularmente para planejamento estratégico dos espaços
  • Configure tabelas de preços específicas para diferentes tipos de eventos e clientes
Dicas de Otimização
  • Use templates de eventos para tipos de eventos recorrentes e economize tempo
  • Configure lembretes automáticos para follow-up de eventos com status "Quente"
  • Revise eventos com status "Perdido" periodicamente para identificar padrões e oportunidades de melhoria
  • Use relatórios do módulo para análise de ocupação e performance dos espaços
  • Treine toda a equipe na utilização consistente do módulo para padronização dos dados
