Consultar Histórico de um Quarto Virtual

Passo a passo para acessar o histórico completo de hospedagens de um quarto específico

O histórico de quarto permite visualizar todas as hospedagens anteriores em um quarto específico, útil para análise de ocupação, manutenção e atendimento a hóspedes.
1 Acessar Hospedagens
2 Configurar Filtros
3 Aplicar Filtros
4 Analisar Resultados

Passo 1: Acessar a Tela de Hospedagens

Vá até o menu lateral do sistema e clique na opção "Hospedagens".

Localize e clique na opção:

Hospedagens

Ou, se preferir, acesse diretamente pelo link:

https://hotel.bitzsoftwares.com.br/sistema/locacao/

Dica: Você pode favoritar o link das Hospedagens no seu navegador para acesso rápido sempre que precisar consultar históricos.

Passo 2: Configurar os Filtros de Consulta

Na página que abrir, utilize os filtros disponíveis para refinar sua busca pelo histórico do quarto.

Filtros de Consulta

até
  • No campo "Quarto": Selecione o quarto desejado para consultar seu histórico específico
  • No campo "Período": Selecione o intervalo de datas que deseja consultar (ex: últimos 30 dias, último ano, etc.)
  • Filtros adicionais: Use os outros filtros disponíveis para refinar ainda mais sua busca, se necessário

    • Passo 3: Aplicar os Filtros e Gerar Resultados

    Após configurar todos os filtros, clique em "Filtrar" para executar a consulta.

    O sistema processará sua solicitação e exibirá todas as hospedagens anteriores de acordo com os filtros aplicados.

    Dicas para uma consulta eficiente:

    • Para uma visão geral, selecione um período maior (ex: últimos 6 meses)
    • Para uma busca específica, reduza o período (ex: últimos 30 dias)
    • Você pode salvar combinações de filtros frequentes, se o sistema permitir
    Filtros aplicados com sucesso! O sistema está processando o histórico do quarto selecionado.

    Informações Disponíveis no Histórico:

    • Dados do hóspede: Nome completo e documento de identificação
    • Datas de estadia: Check-in e check-out exatos
    • Duração: Número de noites da hospedagem
    • Status: Concluída, ativa ou cancelada
    • Valor total: Faturamento gerado pela hospedagem
    • Observações: Anotações especiais registradas durante a estadia

    Resumo do Processo:

    • Acesse "Hospedagens" pelo menu lateral ou link direto
    • Selecione o quarto desejado no campo de filtro
    • Defina o período de consulta (data inicial e final)
    • Clique em "Filtrar" para executar a consulta
    • Analise os resultados na tabela de histórico
    • Exporte os dados se necessário para análise externa

    Usos do histórico de quarto:

    • Análise de ocupação: Verifique padrões de uso do quarto
    • Manutenção preventiva: Base para programar limpezas técnicas
    • Atendimento ao hóspede: Histórico para casos de reclamações ou elogios
    • Auditoria: Verificação de movimentações no quarto
    Consulta concluída! Agora você tem acesso completo ao histórico de hospedagens do quarto selecionado.
