Como Transferir um Acompanhante para Outro Quarto

Processo correto para mover acompanhantes entre diferentes UHs

Não é possível fazer a transferência direta de um acompanhante para outro quarto pelo sistema. Para realizar esse processo corretamente, é necessário seguir os passos abaixo:
1 Validar consumos
2 Check-out do acompanhante
3 Verificar pendências
4 Incluir no novo quarto
5 Processo concluído

Passo 1: Validar todos os consumos do quarto original

Antes de iniciar a transferência, é essencial verificar se existem consumos vinculados ao acompanhante que será transferido.

Quarto 101 - STDC

Hospedagem atual:

  • João Silva
    Titular
  • Maria Santos
    Acompanhante
Quarto 102 - STDC

Quarto de destino:

  • Carlos Oliveira
    Titular

Verifique os consumos vinculados a Maria Santos (acompanhante):

Descrição Responsável Valor Ação Necessária
Café da Manhã Maria Santos R$ 25,00
Lavanderia Maria Santos R$ 45,00
Água Mineral João Silva R$ 8,00 Nenhuma ação necessária

Importante: Caso algum lançamento esteja vinculado ao acompanhante que será transferido, realize uma das seguintes ações:

  • Baixa do consumo - Se o consumo já foi pago/solucionado
  • Exclusão - Se foi um lançamento incorreto
  • Transferência de responsabilidade - Mover para o titular ou outro hóspede

Passo 2: Check-out do acompanhante no quarto atual

Após garantir que não há pendências de consumo, acesse a aba "Acompanhantes" dentro da hospedagem do quarto atual.

Quarto 101 - Acompanhantes
  • João Silva
    Titular
  • Maria Santos
    CPF: 123.456.789-00

Clique na opção "Check-out" ao lado do nome do acompanhante que será transferido.

Observação: O sistema solicitará confirmação para realizar o check-out do acompanhante. Isso não afetará a hospedagem principal, apenas removerá o acompanhante da lista atual.

Passo 3: Verificar pendências antes de finalizar

Ainda na aba "Acompanhantes", o sistema permite visualizar se existem pendências de consumo antes de finalizar o check-out.

Verificação de Pendências

  • Consumos pagos: Nenhum consumo pendente vinculado ao acompanhante
  • Documentação: Documentos do acompanhante estão em dia
  • Chaves e acessórios: Nenhum item pendente de devolução
  • Atenção: Após o check-out, o acompanhante não terá mais acesso ao quarto 101

Esta verificação ajuda a evitar inconsistências no sistema e garante que o acompanhante possa ser transferido corretamente sem deixar pendências no quarto de origem.

Dica: Se o sistema mostrar pendências de consumo, retorne ao Passo 1 para resolvê-las antes de prosseguir com o check-out.

Passo 4: Incluir o acompanhante no novo quarto

Depois de realizar o check-out do acompanhante, acesse a hospedagem do novo quarto de destino.

Quarto 102 - STDC
  • Carlos Oliveira
    Titular da hospedagem
    Titular

Siga o processo normal de inclusão de um novo acompanhante:

  1. Preencha os dados pessoais do acompanhante (nome completo, CPF, data de nascimento)
  2. Informe os dados do documento de identificação
  3. Adicione informações de contato (telefone, e-mail)
  4. Faça o upload da foto, se necessário
  5. Clique em "Salvar" para incluir o acompanhante na nova hospedagem

Passo 5: Processo concluído

Após seguir todos os passos corretamente, o acompanhante estará transferido para o novo quarto.

Transferência realizada com sucesso! Esta é a forma correta e segura de transferir um acompanhante entre quartos dentro do sistema.

Resumo do processo:

  • Valide consumos no quarto de origem
  • Faça check-out do acompanhante na aba "Acompanhantes"
  • Verifique pendências antes de finalizar
  • Cadastre como novo acompanhante no quarto de destino
  • Confirme a transferência está completa

Lembrete importante: Não existe opção de "transferir" diretamente um acompanhante entre quartos. Sempre é necessário fazer o check-out no quarto atual e novo cadastro no quarto de destino.
