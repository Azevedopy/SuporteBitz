Como Alterar Status do Quarto para Limpo

Guia completo para alterar o status de um quarto de "Sujo" para "Limpo" no sistema Bitz

Com um quarto com status de sujo e precisamos alterar o status de um quarto para "Limpo", siga o passo a passo abaixo:

1 Acesso aos Painéis
2 Visualizar Quartos
3 Selecionar Quarto Sujo
4 Liberar Quarto
5 Confirmação e Limitações

Passo 1: Acesso aos Painéis

No menu lateral, acesse a opção Painéis.

Localização do Menu

O menu "Painéis" está localizado na barra lateral esquerda do sistema Bitz. É a principal interface para visualização e gerenciamento dos quartos.

Acesso direto: PAINEL DE UH'S

Passo 2: Visualizar a lista de quartos

Após acessar os painéis, visualize a lista de quartos disponíveis e em curso.

Visualização dos Quartos

O painel exibe todos os quartos do hotel com seus respectivos status. Os quartos são apresentados em formato de cards ou lista, mostrando informações como:

  • Número do quarto
  • Status de ocupação
  • Situação atual (limpo, sujo, interditado)
  • Informações da hospedagem

Passo 3: Selecionar o quarto com status sujo

Selecione o quarto cujo status esteja em amarelo representando que está sujo.

Quarto Sujo

Clique para Selecionar

Identificação Visual

Os quartos com status "Sujo" geralmente são destacados com cor amarela ou laranja no sistema, diferenciando-se dos quartos "Limpos" (geralmente verde) ou "Interditados" (vermelho).

Passo 4: Localizar e clicar no ícone do cadeado

Localize o ícone de um cadeado para fazer a liberação do quarto de sujo para limpo.

Ícone do Cadeado

O ícone do cadeado representa a funcionalidade de liberação/alteração de status do quarto. Ele pode aparecer como:

  • Cadeado fechado: indica que o quarto está bloqueado/sujo
  • Cadeado aberto: indica que o quarto está liberado/limpo
  • Posicione o mouse sobre o ícone do cadeado
  • Clique no ícone para abrir a opção de liberação
  • Aguarde o sistema processar a solicitação

    • Passo 5: Confirmação da alteração e limitações

    O sistema exibirá uma mensagem de confirmação da disponibilização.

    Mensagem de Confirmação

    O sistema apresentará uma mensagem perguntando se você realmente deseja alterar o status do quarto. Confirme a alteração clicando em "Sim" ou "Confirmar".

    Limitação Importante

    Durante uma hospedagem ativa, não é possível alterar o status do quarto para "Limpo" diretamente. Nesse caso, o sistema permite apenas a interdição do quarto, com a justificativa de limpeza.

    Quartos em Hospedagem

    Quando um quarto está em uso (hospedagem ativa):

    • Não é possível mudar para "Limpo"
    • Apenas interdição é permitida
    • Justificativa de limpeza é obrigatória
    Status alterado com sucesso! O quarto agora está com status "Limpo" e disponível para novas hospedagens.
