Mensagem de erro: "Não há tarifa cadastrada para a quantidade de pessoas informada"

Como resolver esse problema passo a passo

Erro: "Não há tarifa cadastrada para a quantidade de pessoas informada"

O que significa essa mensagem?

O sistema não encontrou nenhuma tarifa registrada para a quantidade exata de pessoas inserida na reserva.

Exemplo: Você tentou fazer uma reserva para 3 adultos, mas no sistema só há tarifas cadastradas para até 2 adultos. Por isso, a reserva não pode ser concluída.

1 Verificar tarifas cadastradas
2 Confirmar capacidade do quarto
3 Cadastrar nova tarifa
4 Conclusão

Passo 1: Verificar as tarifas cadastradas

Você precisa checar se já existe um valor configurado no sistema para a quantidade de pessoas da reserva. Há duas formas de fazer isso:

Opção 1: Acessar a tela Tarifas NOVO

Vá no menu lateral:

Cadastros > Operacional > Calendário de Tarifas > Tarifas NOVO

Ou utilize o link: TARIFAS

Opção 2: Usar o Mapa Tarifário

No menu lateral direito, clique em Mapa Tarifário

Ou acesse diretamente pelo link: MAPA TARIFÁRIO

Após essa verificação, vamos validar se o quarto comporta essa quantidade de pessoas.

Passo 2: Confirmar se o quarto comporta a quantidade de pessoas

Antes de cadastrar uma nova tarifa, é necessário verificar se o quarto permite essa quantidade de hóspedes.

Acesse: Cadastros > Operacional > Classificação de UH

Ou use o link direto: CLASSIFICAÇÕES DE UH

Clique em "Editar" na classificação do quarto que você deseja usar e verifique os seguintes campos:

Máximo de adultos
Limite de adultos permitidos
Máximo de crianças
Limite de crianças permitidas
Número máximo de pessoas
Total máximo de hóspedes

Importante: Se a quantidade de pessoas que você quer colocar na reserva for maior do que os limites acima, o sistema não permitirá cadastrar a tarifa. Nesse caso:

  • Escolha outra classificação de quarto
  • Ajuste a quantidade de hóspedes
  • Se possível, ajuste esses campos de limite

Passo 3: Cadastrar uma tarifa para uma quantidade de pessoas específica

Agora que você confirmou que a quantidade de pessoas é permitida para o quarto, siga os passos para cadastrar o valor:

No sistema, clique na aba lateral direita e localize a opção Mapa Tarifário

Ou acesse diretamente pelo link: MAPA TARIFÁRIO

  1. No canto superior da tela, clique em "Atualizar Preço"
  2. Informe a data inicial e final em que essa tarifa vai valer
  3. Escolha qual tarifa você quer alterar
  4. O sistema mostrará todos os quartos e quantidades de pessoas configuradas
  5. Identifique o campo referente à quantidade de pessoas que você precisa (ex: 3 adultos)
  6. Se o campo estiver vazio ou com valor 0, digite o novo valor da diária
  7. Clique no botão de salvar (ícone de disquete no canto inferior)

Passo 4: Conclusão

Após seguir todos os passos e salvar a nova tarifa, o sistema estará pronto para aceitar reservas com a quantidade de pessoas que estava causando o erro.

Resumo do processo:

  • Verifique se existe tarifa para a quantidade de pessoas
  • Confirme se o quarto suporta essa quantidade
  • Cadastre a tarifa específica se necessário
  • Conclua a reserva normalmente
Problema resolvido! O sistema já poderá aceitar reservas com a nova quantidade de pessoas, aplicando corretamente a tarifa cadastrada.

Dica: Mantenha seu mapa tarifário sempre atualizado para evitar esse tipo de erro no futuro. Certifique-se de cadastrar tarifas para todas as combinações de ocupação possíveis em cada tipo de quarto.
