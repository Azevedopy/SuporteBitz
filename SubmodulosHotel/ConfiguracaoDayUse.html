Configurações de Day Use

Configure os parâmetros essenciais para o funcionamento do Day Use no seu hotel

No menu lateral esquerdo, acesse o campo de Day Use e selecione a opção "Configurações de Day Use". Selecione a opção de editar para alterar as configurações conforme as necessidades do seu estabelecimento.

Passo 1: Como Acessar as Configurações

Para acessar e configurar as opções do Day Use, siga o caminho no sistema:

Caminho de Acesso

Menu lateral → Day Use → Configurações de Day Use

Após acessar a tela de configurações, clique no botão "Editar" para modificar as configurações disponíveis.

Passo 2: Capacidade Total de Pessoas

Capacidade Total de Pessoas (Hóspedes e Visitantes)

O que é: É o limite máximo de pessoas permitidas simultaneamente no espaço do Day Use. Inclui tanto hóspedes do hotel quanto visitantes externos.

85% da Capacidade
Para que serve?
  • Controlar lotação do parque/piscinas/área de lazer
  • Impedir vendas acima da capacidade máxima permitida
  • Ajudar na gestão da operação e segurança do estabelecimento
  • Garantir conformidade com normas de segurança e bombeiros
Como funciona no sistema?

Quando a capacidade é atingida, o sistema bloqueia automaticamente novas reservas ou vendas de Day Use para aquela data. O sistema usa essa configuração para:

  • Monitorar o fluxo diário de pessoas no estabelecimento
  • Exibir alertas visuais quando a capacidade está próxima do limite
  • Gerar relatórios de ocupação e capacidade utilizada
  • Atualizar em tempo real a disponibilidade de vagas

Boa Prática

Defina a capacidade total considerando não apenas a área física, mas também fatores como:

  • Metragem quadrada das áreas de lazer
  • Quantidade de equipamentos disponíveis
  • Capacidade do restaurante/buffet
  • Normas de segurança e bombeiros

Passo 3: Utiliza Pulseira

Utiliza Pulseira

O que é: Indica se o estabelecimento usa pulseiras de identificação para controlar o acesso dos visitantes Day Use.

DAY USE VIP
Para que serve?
  • Identificar quem está autorizado a usar as áreas do Day Use
  • Ajudar a equipe a controlar entradas e permanência nas áreas comuns
  • Diferenciar tipos de acesso (ex.: adulto, criança, vip) — dependendo das regras do hotel
  • Rastrear o tempo de permanência dos visitantes no estabelecimento
No sistema:

Ao marcar que utiliza pulseira, o sistema habilita automaticamente as seguintes funcionalidades:

  • Registro de entrega de pulseira - Controle de quais visitantes receberam pulseira
  • Controle de quantidade usada no dia - Monitoramento do estoque de pulseiras
  • Relatórios de acesso - Documentação de quem acessou as áreas e quando
  • Integração com leitores de código - Para controle rápido de acesso
  • Personalização por cor - Diferentes cores para diferentes tipos de acesso

Controle de Acesso

Garante que apenas pessoas autorizadas tenham acesso às áreas de lazer.

Análise de Dados

Fornece dados precisos sobre o fluxo de visitantes e tempo médio de permanência.

Segurança

Aumenta a segurança do estabelecimento controlando quem está nas áreas comuns.

Controle Financeiro

Evita que pessoas não pagas acessem as áreas de lazer do hotel.

Passo 4: Resumo das Configurações

Configurações Essenciais do Day Use

  • Acesse as configurações: Menu lateral → Day Use → Configurações de Day Use
  • Clique em "Editar" para modificar as configurações
  • Configure a Capacidade Total: Defina o limite máximo de pessoas permitidas simultaneamente
  • Defina se utiliza pulseira: Ative esta opção se usar pulseiras para controle de acesso
  • Salve as alterações para que as novas configurações entrem em vigor

    • Salvar Alterações

    Não se esqueça de salvar as alterações para que as novas configurações sejam aplicadas.

    Histórico

    O sistema mantém histórico de todas as alterações realizadas nas configurações.

    Atualização Imediata

    As alterações são aplicadas imediatamente após o salvamento.

    Acesso Restrito

    Apenas usuários com permissão podem alterar as configurações do Day Use.

    Configurações do Day Use ajustadas com sucesso! O sistema agora está pronto para gerenciar as reservas e acesso de visitantes com base nas configurações definidas.

    Recomendações Finais

    • Revise periodicamente as configurações conforme a sazonalidade
    • Ajuste a capacidade conforme mudanças na estrutura física
    • Documente as alterações realizadas nas configurações
    • Configure alertas quando a capacidade estiver próxima do limite
