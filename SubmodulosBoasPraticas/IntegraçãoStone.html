2. Parceria com Stone

Existem diferentes formas de estabelecer a parceria com a Stone, dependendo do tipo de transação que o hotel deseja realizar.

Indicação de Cliente para Stone

Para indicar um cliente à Stone, preencha o formulário oficial:

https://app.pipefy.com/public/form/JvcC9XwE

OU o cliente pode realizar cadastro direto conforme abaixo:

Máquina Física

Integração Connect

Transações com POS físico no hotel

Cadastro Cliente Stone

Importante: Selecionar terminal "Aluguel" no cadastro

Transações Online

Integração Gateway

Pagamentos online (motor, links)

Cadastro Pagar.me

Importante: Selecionar terminal "E-commerce" no cadastro

Responsabilidades do Cliente
  • Negociação de taxas: Diretamente com a Stone
  • Cadastro completo: Seguir os links específicos
  • Seleção correta: Terminal "Aluguel" ou "E-commerce"
  • Documentação: CNPJ e dados empresariais

Alternativa: Channex

Quando o cliente não quiser adquirir Stone por causa das taxas:

É possível trocar o motor da Bitz pelo motor da Channex. Usando a Channex e seu motor de reservas é possível realizar pré-pagamentos sem ter nenhuma integração bancária, ou seja, sem taxas bancárias.

Funcionamento: O hotel recebe os dados do cartão do hóspede como pré-pagamento e ao chegar no hotel o hóspede passará o cartão na máquina do hotel para confirmar o pagamento.

3. Liberação da Integração na Bitz

Após o cadastro na Stone, é necessário solicitar a liberação da integração no sistema Bitz.

Contato Comercial Bitz

O cliente deve contatar nosso comercial solicitando o upsell da integração com a Stone diretamente com:

Luiza
(46) 99107-2468

Cliente Cadastrado

Stone ou Pagar.me

Contato Comercial

Solicitação de upsell

Liberação

Equipe Bitz ativa integração

Informações Críticas ao Cliente

Cliente precisa cadastrar a chave PIX dentro do app da Stone para poder liberar o módulo de PIX.

Sem a chave PIX configurada no app da Stone, o módulo de pagamentos via PIX não funcionará no sistema Bitz.

Processo de Liberação
  1. Cliente realiza cadastro na Stone/Pagar.me
  2. Cliente contata comercial Bitz (Luiza)
  3. Solicitação formal de upsell
  4. Equipe Bitz valida cadastro
  5. Liberação técnica da integração
  6. Configuração no sistema Bitz

4. Processo de Implantação

O processo de implantação varia conforme o cliente já seja ou não cliente da Stone.

Cenário 1: Cliente Já é Cliente Stone

Connect (Máquina Física)

Modelo de captura: Aluguel

Ativação: Partnerhub

Responsável: Deyse

Gateway (Online)

Modelo de captura: E-commerce

Ativação: Pagar.me

Responsável: Alan
Processo para Cliente Existente
  1. Criar ticket com macro "Ativação Stone"
  2. Responsável: Deyse (Connect) ou Alan (Gateway)
  3. Informações necessárias: StoneCode e CNPJ
  4. Plataforma: Ativar API no Partnerhub ou Pagar.me
  5. Campo: 'Ativação de lojista'
  6. Passar dados para produto (Giovani) para liberação no painel
Download do Aplicativo
  • Aplicação será baixada em até 3 horas após liberação
  • Terminal deve estar ligado para download
  • POS conectado ao Wi-fi ou 3G
  • Verificar atualizações em: Menu → Ajustes → Verificar atualizações
Cenário 2: Cliente Novo na Stone

O cliente deve ser indicado pela Bitz, através do formulário de indicação:

Formulário de Indicação

A Stone entrará em contato para negociar taxas

Formulário de Indicação Stone
  • Cliente preenche formulário de indicação
  • Stone entra em contato para negociar taxas
  • Cadastro fica no dashboard do Partnerhub
  • Status no kanban: coluna "credenciado"
  • Após chaves disponíveis, solicitar liberação a Giovani

    • Observações Importantes

    • Link de pagamento e PIX: Obrigatório cadastro de CPF e telefone (exigência Stone)
    • Taxas diferentes por operadora: Cadastrar operadoras separadamente
    • Erro na ativação: Solicitar reinício da máquina (desligar 10 segundos)

    5. Configuração no Sistema Bitz

    Após a liberação técnica, configure o sistema Bitz para utilizar a integração com Stone.

    Passo 1: Habilitar Gateway Stone

    Na configuração de módulos adicionais, habilitar o gateway da Stone:

    Módulos Adicionais
    Gateway Stone
    Integração com Stone para pagamentos
    Passo 2: Configurar Forma de Pagamento PIX

    Criar uma forma de pagamento padrão para o PIX da Stone:

    Cadastro de Formas de Pagamento
    PIX Stone
    Tipo: Outros
    Status: Ativo
    Observação: Internamente continua sendo um acerto em "Outros"
    Passo 3: Configurar Motor de Reservas

    Nos parâmetros do motor deve ser informada a forma de pagamento padrão para a Stone:

    Parâmetros do Motor de Reservas
    Forma de Pagamento Padrão Stone:
    PIX Stone
    Passo 4: Configurar Terminal de Pagamento

    Para recebimentos na comanda do hotel, configure um terminal de pagamento:

    Cadastro de Terminal de Pagamento
    Tipo: Stone
    Pode ser o mesmo para PIX: Sim
    Configuração Operadoras:
    Marcar opção "STONE" nas operadoras cadastradas
    Funcionamento no Quarto

    Ao informar a operadora que usa Stone no quarto:

  • Habilitará o campo do terminal
  • Ao adicionar, mostrará a tela de pagamento da Stone
  • Processamento automático do pagamento

    • Atenção: Configuração da Máquina

    • Máquina Smart: Função de pedidos ativada (Ajustes)
    • Status: Máquina deve estar "aguardando pedido"
    • Conexão: Internet estável (Wi-fi preferencialmente)

    6. Troubleshooting e Problemas Comuns

    Soluções para problemas comuns que podem ocorrer durante a implantação e uso da integração.

    Problema: Falha ao Configurar Aplicativo com API POSConnect

    Sintoma: Erro ao configurar o aplicativo com a API POSConnect.

    Solução 1: Atualizar Maquininha

    Acessar "Configurações/Ajuste" → "Atualizar"

    Garantir conexão estável à internet, preferencialmente Wi-Fi.

    Solução 2: Verificar Vínculo

    Certificar que o serial do POS está vinculado ao Stone Code de teste.

    Caso não esteja, preencher formulário de vínculo.

    Erro Específico

    "COULD_NOT_ACTIVATE_WITH_ACCEPTOR_CONFIGURATION_UPDATE_DATA_NULL"

    Indica que o serial não foi vinculado corretamente.

    Procedimento de Reinicialização

    Quando ocorrer erro na ativação com máquina física:

    1. Solicitar reinício da máquina
    2. Desligar completamente por 10 segundos
    3. Ligar novamente e aguardar inicialização
    4. Verificar conexão com internet
    5. Tentar ativação novamente
    Problemas de Conexão
    • Verificar internet: Wi-fi ou 3G estável
    • Reiniciar POS: Desligar e ligar após 30 segundos
    • Atualizar sistema: Menu → Ajustes → Verificar atualizações
    • Testar conexão: Navegar na internet pelo POS
    Problemas com Pagamentos

    Pagamento Não Processado

    Verificar se a máquina está em modo "aguardando pedido"

    Conferir se função de pedidos está ativada

    PIX Não Gerado

    Verificar se chave PIX está cadastrada no app Stone

    Confirmar cadastro de CPF e telefone no sistema

    Operadora Não Reconhecida

    Conferir se operadora está marcada como "STONE"

    Verificar cadastro completo da operadora

    Se o problema persistir, compartilhe detalhes do erro com nossa equipe de suporte para que possamos ajudar melhor!
