Integração Rede

Guia completo para configuração do gateway de pagamento

Esta documentação orienta sobre o processo completo para habilitar a integração com a Rede, desde a coleta das credenciais até a configuração no sistema Imperium.

Pré-requisito

O cliente deve possuir um cadastro ativo na Rede para prosseguir com a integração.

Se o cliente ainda não possui cadastro, é necessário criar uma conta no site da Rede antes de iniciar o processo.

1. Dados Necessários para Integração

Para habilitar a integração, é preciso coletar duas informações fundamentais dentro do painel administrativo da Rede.

Credenciais Obrigatórias
Merchant ID: ID_DA_LOJA_EXEMPLO
Merchant Key: CHAVE_SECRETA_EXEMPLO

O que são estas credenciais?

Merchant ID (ID da Loja): Identificador único da loja dentro da plataforma Rede.

Merchant Key (Chave da Loja): Chave de segurança para autenticação das transações.

Estes dados são sensíveis e devem ser tratados com confidencialidade.

Atenção

Não compartilhe estas credenciais por e-mail não seguro ou mensagens não criptografadas. Utilize canais seguros para envio ao Giovani.

2. Como Localizar as Credenciais

Siga os passos abaixo para encontrar as credenciais dentro do painel administrativo da Rede.

  1. Acesse o painel ADMIN da Rede com as credenciais do cliente.
  2. No menu principal, navegue até Configurações > Geral > Dados da Loja.
  3. Localize os campos na seção de informações da loja.
  4. Copie os valores dos campos:
    • ID da Loja (Merchant ID)
    • Chave da Loja (Merchant Key)
  5. Anote ou copie estes dados para envio.

Não encontrou as credenciais?

Se você não encontrar os campos Merchant ID e Merchant Key na seção "Dados da Loja", verifique:

  • Se você possui permissões de administrador
  • Na seção "API" ou "Integrações"
  • Nas configurações de "Segurança" ou "Chaves de API"
  • Entre em contato com o suporte da Rede se necessário

3. Encaminhamento dos Dados

Após coletar as credenciais, é necessário encaminhá-las para o responsável pela integração.

Responsável pela Integração

Giovani

Responsável pelo cadastro e configuração da integração

Envie as credenciais para o e-mail corporativo

Como encaminhar os dados

Siga estas recomendações para envio seguro das credenciais:

  1. Envie as duas informações (Merchant ID e Merchant Key) juntas
  2. Identifique claramente qual é cada informação
  3. Informe o nome do cliente/estabelecimento
  4. Use canais seguros (e-mail corporativo, sistema interno)
  5. Confirme o recebimento com o Giovani

Att.

Tempo de Processamento

Após o envio das credenciais, o processo de cadastro e configuração leva aproximadamente 24 a 48 horas úteis para ser concluído pelo Giovani.

Você será notificado quando a integração estiver pronta para ser configurada no sistema.

4. Configuração no Sistema Imperium

Após o cadastro realizado pelo Giovani, siga os passos abaixo para habilitar a integração no sistema.

Cadastro > Operacional > Parâmetros de Pagamento
  1. Acesse o menu conforme indicado acima
  2. Clique no botão + (Adicionar)
  3. Preencha os campos conforme as opções que serão utilizadas
  4. Salve as informações

Dicas de Configuração

Para múltiplas configurações: Crie um parâmetro para cada tipo (Cartão de Crédito, Débito, etc.)

Taxas: Consulte com o cliente qual a taxa cobrada pela Rede para configurar corretamente.

Parcelas: Defina conforme acordo comercial com a operadora.

Verificação

Após salvar, a opção cadastrada ficará disponível para seleção dentro do módulo Motor de Reservas.

Verifique se o parâmetro aparece na listagem antes de prosseguir para o próximo passo.

5. Configuração no Motor de Reservas

Finalize a configuração habilitando a integração no Motor de Reservas.

Cadastros > Operacional > Parâmetros Motor de Reservas
  1. Acesse o menu conforme indicado acima
  2. Localize a configuração do Motor de Reservas do estabelecimento
  3. Na seção de pagamentos, selecione a opção Gateway de Pagamento
  4. Informe a Rede (Rede)
  5. Selecione o parâmetro criado anteriormente
  6. Salve as configurações

Teste da Integração

Após concluir a configuração, realize os seguintes testes:

  1. Crie uma reserva de teste no Motor de Reservas
  2. Tente realizar um pagamento com valor simbólico (ex: R$ 1,00)
  3. Verifique se a transação é processada corretamente
  4. Confirme se o status da reserva é atualizado
  5. Verifique os logs de transação no sistema
Integração configurada com sucesso! O sistema está pronto para processar pagamentos através da Rede.
