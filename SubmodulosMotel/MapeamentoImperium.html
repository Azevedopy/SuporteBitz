Tutorial de Mapeamento do Sistema

BITZ Softwares - Configuração de Rede e Acesso

Este tutorial guiará você através do processo completo de mapeamento do sistema BITZ Softwares, desde a configuração do servidor até a conexão das estações cliente. Siga cada passo cuidadosamente para garantir um funcionamento perfeito.

1. Servidor: Liberar a Pasta

Primeiro passo para configurar o servidor: liberar a pasta principal do sistema para compartilhamento.

Procedimento
  • Busque a pasta "BITZ Softwares" no disco "C".
  • Clique com o botão direito sobre ela e vá em "Propriedades".

2. Servidor: Compartilhamento Avançado

Configurar o compartilhamento avançado da pasta para permitir acesso em rede.

Compartilhamento Avançado
  • Na janela de propriedades, selecione a aba "Compartilhamento".
  • Clique no botão "Compartilhamento Avançado...".
  • Marque o check-box "Compartilhar a pasta".
  • Em seguida, clique no botão "Permissões".

3. Servidor: Permissões de Compartilhamento

Definir permissões de compartilhamento para garantir acesso adequado à pasta.

Controle Total para Todos
  • No grupo "Todos", selecione a caixa "Controle total" na coluna 'Permitir'.
  • Clique em "Aplicar" e depois em "OK".

4. Servidor: Configuração de Segurança

Configurar permissões de segurança para completar o acesso à pasta.

Segurança da Pasta
  • Volte até as propriedades da pasta e, desta vez, acesse a aba "Segurança".
  • Clique no botão "Editar...".
  • Na janela de permissões, clique em "Adicionar...".
  • Digite o nome do usuário "Todos" na caixa de texto.
  • Clique em "Verificar nomes" (o nome ficará sublinhado se estiver correto).
  • Clique em "OK".
  • Selecione o usuário "Todos" que acabou de adicionar na lista.
  • Marque a caixa "Controle total" na coluna 'Permitir'.
  • Clique em "Aplicar" e depois em "OK".

5. Cliente: Mapear Unidade de Rede

Configurar a estação cliente para acessar a pasta compartilhada do servidor.

Mapeamento de Rede
  • Acesse o executar do Windows (Pressione Windows+R).
  • Digite o IP da máquina servidora que deseja mapear. Exemplo: \\\\192.168.1.10
  • Clique em OK.
  • Localize a pasta "BITZ Softwares" compartilhada.
  • Clique com o botão direito nela e selecione a opção "Mapear unidade de rede...".
  • Na janela de mapeamento, selecione a unidade "B:" na lista de unidades.
  • Clique em "Concluir".

6. Cliente: Instalação do InstantClient

Instalar o Oracle InstantClient necessário para conexão com o banco de dados.

Procedimento de Instalação
  • Após mapear a unidade, vá até a pasta "C:/BITZ Softwares" > "Utilitários".
  • Execute o instalador "instantclient".
  • Prossiga a instalação até a tela de seleção de tarefas adicionais.
ATENÇÃO

Marque a opção "Adicionar o diretório do software na variável de sistema PATH" durante a instalação!

7. Cliente: Configurar TNSNAMES.ORA

Configurar o arquivo TNSNAMES para conexão com o banco de dados Oracle.

Configuração do Arquivo
  • Após a instalação, acesse a pasta "C:\instantclient".
  • Abra o arquivo "TNSNAMES" com o Bloco de Notas.
  • Altere os campos <SERVIÇO> e <SID> para "XE".
  • Altere o campo <IP> para o IP da máquina servidora (ex: 192.168.1.10).
Antes (DE):
# Serviço: <SERVIÇO>
<SERVICO> =
  (DESCRIPTION =
    (ADDRESS_LIST =
      (ADDRESS = (PROTOCOL = TCP)(HOST = <IP>)(PORT = 1521))
    )
    (CONNECT_DATA =
      (SERVICE_NAME = <SID>)
    )
  )
Depois (PARA):
# Serviço: XE
XE =
  (DESCRIPTION =
    (ADDRESS_LIST =
      (ADDRESS = (PROTOCOL = TCP)(HOST = 192.168.1.10)(PORT = 1521))
    )
    (CONNECT_DATA =
      (SERVICE_NAME = XE)
    )
  )

8. Variáveis de Ambiente

Configurar variáveis de ambiente do sistema para funcionamento correto do InstantClient.

Configuração das Variáveis
  • Em "Meu Computador" ou "Este Computador", clique com o botão direito.
  • Selecione "Propriedades".
  • No menu lateral ou na janela que abrir, clique em "Configurações avançadas do sistema".
  • Na aba "Avançado", clique no botão "Variáveis de Ambiente..." na parte inferior.
  • Na seção "Variáveis de usuário" (parte superior), clique em "Novo...".
  • Nome da variável: TNS_ADMIN
  • Valor da variável: C:\instantclient
  • Clique em OK.
  • Na seção "Variáveis de sistema" (parte inferior), procure a variável "Path" e clique em "Editar".
  • Mova o diretório "C:\instantclient" para o topo da lista.

9. Conclusão e Teste

Finalizar o processo de mapeamento e testar o acesso ao sistema.

Finalização
  • O mapeamento foi finalizado.
  • Acesse a pasta "BITZ Softwares" mapeada.
  • Crie um atalho do "Imperium.exe" para a Área de Trabalho.
  • Execute o sistema e teste o acesso.
Sucesso! As configurações de mapeamento e variáveis de ambiente foram concluídas. O sistema está pronto para uso.
