Identifique o tipo de tabela configurada utilizando a função "Cotação de Hospedagem" localizada na barra superior do sistema Imperium.

Processo de Diagnóstico

Siga estes passos para identificar a tabela configurada:

  1. Clique no botão "Cotação de Hospedagem" na barra superior
  2. Selecione o quarto desejado na lista
  3. Altere horários de entrada/saída conforme necessário
  4. Ajuste quantidade de casais se for o caso
  5. Escolha o tipo de hospedagem (Locação, Pernoite ou Diária)
  6. Clique em "Calcular" para ver valores e tabela vigente

Importante

O campo "Tabela Vigente" mostrará qual a forma que o quarto está calculando os valores. Esta informação é crucial para saber onde fazer os ajustes.

3. Tempo Especial e Configuração de Adicionais

Configure tempos especiais e regras para horas adicionais no cadastro do quarto.

Tempo Especial

O campo "Tempo Especial" define minutos para tempo promocional. Só terá validade para quartos que não estão com tabelas de valores definidos.

Exemplo Prático

Configuração:

  • Locação configurada para 1 hora
  • Especial de 30 minutos

Funcionamento: Caso o cliente fique por 20 minutos, irá cobrar o valor do tempo especial e não o valor de 1 hora cheia.

Quarto 101 - Luxo
Tempo Especial (Min):
Configuração de Adicionais

O sistema utiliza a quantidade de horas definida no cadastro do quarto. Quando passar das horas configuradas, inicia a contagem do campo "Tolerância (Min)".

Exemplo: Tolerância

Configuração:

  • Locação configurada para 1 hora
  • Tempo de tolerância 15 minutos

Cenário 1: Cliente ficou 1:12 hrs → Cobra somente 1 hora

Cenário 2: Cliente ficou 1:16 hrs → Cobra hora adicional

Exemplo: Tempo Adicional

Configuração:

  • Locação 1 hora
  • Tolerância 15 minutos
  • Tempo Adicional 30 minutos

Funcionamento:

Caso o cliente fique 1:16 minutos → cobra adicional para +30 minutos

Com essa inclusão, sistema considera: locação + adicional + tolerância

Resultado: Cobra segunda hora adicional a partir de 1:46 mins

Como Funciona

Os Valores por Período podem ser utilizados de duas formas:

  1. Em tabelas de preços: Para valores diferentes em horários específicos
  2. Diretamente no quarto: Campo "Valores Período" no cadastro do quarto

Requisito

O Tipo Acerto Financeiro precisa estar com a opção "Período" para ser efetiva a configuração informada no campo Valores Período.

Acesso e Configuração
  1. Acesse pela lupa ou CTRL + F → "Valores por Período"
  2. Clique em "Incluir" para nova tabela
  3. Preencha os campos:
    • Descrição: Nome da tabela
    • + (Símbolo positivo): Incluir linha
    • - (Símbolo negativo): Remover linha
    • Sequência: Ordem de cobrança
    • Tempo da Negociação: Tempo do período
    • Valor do Período: Valor cobrado
  4. Clique em "Salvar" na barra superior
  5. Horas adicionais: cobradas do campo "Vlr Hora Adic." do quarto
Edição de Tabela Existente

Para editar uma tabela já cadastrada:

  1. Acesse a tela "Valores por Período"
  2. Clique 2x sobre o nome da tabela desejada
  3. Faça as alterações necessárias
  4. Clique fora da tabela ou pressione TAB para efetivar
  5. Clique em "Salvar" para confirmar
Adicionais por Período

Os "Adicionais por período" também podem ser utilizados em cadastro de tabelas valores e/ou cadastro dos quartos, para definir valores e períodos diferentes do inserido no cadastro do quarto.

Alteração para Diária

Caso a configuração seja voltada para diária, basta alterar a opção "Tipo de Tabela" dentro da tabela para "Diária". Desta forma, selecionando o quarto no campo Quartos, você estará salvando a tabela como diária.
